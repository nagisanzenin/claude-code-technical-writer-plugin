# Technical Writer Plugin for Claude Code

Complete documentation -- API reference, dev guides, operational docs, and Docusaurus site scaffold.

## Installation

```
/plugin marketplace add nagisanzenin/claude-code-plugins
/plugin install technical-writer@nagisanzenin
```

## Features

- API reference documentation
- Developer guides and tutorials
- Operational and runbook docs
- Docusaurus documentation site scaffold
- Changelog and release notes automation
